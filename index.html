Questions you're probably asking

Is VALSYNC really free? How do you make money?
We don't. VALSYNC has no revenue model. No ads, no subscriptions, no data sales, no "premium" tier. It exists because its contributors want a companion app that respects players. If that ever changes, it'll be open-source and transparent — not a dark-pattern pivot.
Will using this get my Riot account banned?
No. VALSYNC uses Riot Games' official API and operates within their developer terms. It does not inject, modify, or interact with the game client in any way. You're reading data you're authorized to access, through the approved API channel.
Does Riot or anyone else see my data?
Your data travels directly between your device and Riot's servers. VALSYNC has no intermediate server, no data pipeline, no backend. The only exception is XMPP chat messages, which pass through Riot's chat infrastructure (the same one the game client uses).
What happens if I uninstall?
Everything VALSYNC stores — encrypted tokens, local replays, cached game data — is removed when you uninstall. There's nothing left behind on any server because there is no server. You can also revoke access from your Riot account management page at any time.
Which regions and devices are supported?
All Valorant regions are supported. VALSYNC currently runs on Android 8.0 and above. An iOS version is not currently available — the app is built natively for Android using Kotlin and Jetpack Compose.
